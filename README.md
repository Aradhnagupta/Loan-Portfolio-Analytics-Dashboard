#  Loan Portfolio Analytics Dashboard

An interactive **Power BI** dashboard built to analyze and monitor loan portfolio performance using **250,000+ loan records**. The project leverages **MS SQL Server**, **Power BI Dataflows**, **DAX**, and **Incremental Refresh** to deliver scalable and data-driven business insights.

---

##  Project Overview

This project demonstrates an end-to-end Business Intelligence solution for analyzing loan portfolio performance. The dashboard enables stakeholders to monitor key lending metrics, identify default risks, and make informed business decisions through interactive visualizations.

---

##  Objectives

- Analyze loan portfolio performance and customer trends.
- Monitor loan approval and default patterns.
- Track key business KPIs through interactive dashboards.
- Ensure data accuracy using validation techniques.
- Improve report performance using Incremental Refresh.
